Google Map plugin will help to embed Google Custom Map in Posts, Pages as shortcode and sidebar as widget.

== Description ==

 Google Map is awesome plugin to add Google Custom map in website.It is fully customizable and can be use as shortcode.
 It is usable using widget and has all possible options to customize.

https://www.youtube.com/watch?v=jzuwBw4j900

### Features of Google Map Plugin:

* **User friendly and easy to use.**
* **Tested with other popular plugins.**
* **Fully Customizable.**
* **Ability to change Latitude, Longitude.**
* **Ability to change zoom level**
* **Easy to add as widget**
* **Ability to use as shortcode in post and Single page**
* **It can be use in single page, single post,  sidebar, footer etc**

You can upgrade Google Map Plugin from [here](http://plugins.svn.wordpress.org/wp-gmap/) to add more features.


== Installation ==

### That's simple:

1. [Download](http://plugins.svn.wordpress.org/wp-gmap/) Google Map Plugin.   
2. Upload the Google Map plugin into plugins directory "wp-content/plugins"
3. Activate the Google Map plugin from admin panel. 
4. Add Google Map widget to your sidebar.   
5. Add Google Map in posts/pages using shortcode.  

== Frequently Asked Questions ==

Have a question? Make a thread in the support forum and we will get back to you soon.

Our Google Map Plugin is user friendly plugin, but here you can see some frequently asked questions that will help you.

= How to get Latitude and longitude?

Go to latlong.net and enter your place name and then click on Find. Then Latitude and longitude will show bellow. Just use it in your map properties.

= How can change the global settings for Google Map plugin =

Click on Google Map in the left side bar, then set the default values.

= How can i see help manual? =
After install the plugin, Click on Google Map Help under Google Map in the left side bar. Then you would be able to see all help manual.

= How to add Google Map in Post =
Go to Posts->Add New then you will see a button named "Add Google Map" beside media button. Click on it and enter your map properties and click on "insert into post", then save it and the map will appear in your desired post. Enjoy!


= How to add Google Map in Pages =
Go to Pages->Add New then you will see a button named "Add Google Map" beside media button. Click on it and enter your map properties and click on "insert into post", then save it and the map will appear in your desired pages. Enjoy!


= How to add Google Map as widget =
Go to =Appearence=->Widget then you will see a widget named "Google Map". Click on it and enter your map properties then save it and the map will appear in your sidebar or footer. Enjoy!
